1.vue-router

最基本的用法就是自定义router然后注入，下面是几个遇到的问题

1.createRouter中history三个模式怎么选

根据vue-router的官网的介绍，开发过程中使用createWebHashHistory比较好，因为可以直接在地址栏中访问

2.如何嵌套页面

在App.vue中，我们通过使用以下方法使用路由页面。

<RouterView> </RouterView>

嵌套页面也类似

{
	path: '...',
	name: "navigation",
	component: NavigationView,
	children: [  
	    {  
	        path: "",  
	        name: "controller",  
	        component: () => ControllerView,  
	    },
	]
}

之后在NavigationView中需要嵌套的地方加上<RouterView>即可
