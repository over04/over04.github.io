从application.properties中使用@Value读取数据为null

最近更新：2025, Jan 16, 00:00
2024, Jan 05
阅读时长: 1 分钟

1.@Value包裹的变量不能为静态

由于SpringBoot整个都是注入式的，因此不能有静态的数据，设定后不会报错但是值为null

2.通过new产生的实例，@Value值为空

SpringBoot是注入式的，因此必须要通过注入的方式来实例化

class Test {
	TokenUtil tokenUtil;
	
	@Autowired  // 这里可以直接注入变量(直接装饰变量，但是不推荐，代码也会给出警告)
	public void setTokenUtil(TokenUtil tokenUtil) {
	    this.tokenUtil = tokenUtil;
	}
}
